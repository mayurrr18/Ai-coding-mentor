# 🚀 AI Coding Practice Mentor

An intelligent, personalized coding assistant that remembers student mistakes and adapts to individual learning styles. Built with Flask and powered by AI-driven personalization.

## ✨ Features

### 🎯 Personalized Learning
- **Adaptive Challenges**: Challenges are generated based on your skill level and weak concepts
- **Mistake Tracking**: The system remembers your past mistakes and helps you avoid them
- **Personalized Hints**: Get tailored hints based on your learning history

### 📚 Comprehensive Challenge Library
- **Multiple Concepts**: Arrays, Strings, Loops, Functions, Dynamic Programming, Data Structures
- **Difficulty Levels**: Beginner, Intermediate, Advanced
- **Various Languages**: Python, JavaScript, Java support
- **Points System**: Earn points for completing challenges

### 🔍 Intelligent Debugging
- **Error Analysis**: Get detailed explanations for syntax, logic, and runtime errors
- **Pattern Recognition**: Detects similar mistakes from your history
- **Language-Specific Hints**: Context-aware suggestions for your chosen language

### 📊 Progress Tracking
- **Learning Path**: Personalized roadmap based on your strengths and weaknesses
- **Statistics Dashboard**: Visualize your progress with charts and metrics
- **Streak System**: Track your daily practice consistency

### 🔐 User Management
- **Secure Authentication**: Password hashing and session management
- **Profile Storage**: Persistent storage of user preferences and history
- **Points & Achievements**: Gamification to keep you motivated
